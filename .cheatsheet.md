# Cheat Sheet

## vim

### window split

- <c-w>s : split horizontally.
- <c-w>v : split vertically.
- <c-w>c : close the active window.
- <c-w>o : close windows except the active one.

### textobject

- iw : inside word
- iW : inside WORD
- is : inside sentence
- ip : inside paragraph
- gf : jump to the file under cursor line
